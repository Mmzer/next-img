# 图片数组顺序加载

const imgs = ['url1', 'url2', 'url3', ...];

按照图片数组顺序队列加载图片（注：加载完一张再加载下一张）

## 使用说明：

首先:

```bash
$ npm install
```

### dev环境

```bash
$ npm run dev
```

或者：
```bash
$ npm run start
```

### mock假数据

```bash
$ npm run mock
```

mock假数据配置方式，参考文件`mock/_mockserver.json`。

### 运行测试

```bash
$ npm run test
```

单元测试写在`test/`文件夹下。

### prod环境(打包，构建)

```bash
$ npm run prod
```
